Placing on a grid line will place vertex on that line
Placing between grid lines will create a new line between them and place vertex on that line
Lines with no vertices are automatically deleted. use ghost verticies for spacing and centering


Keybindings
lmb/q -> Place Vertex  
mmb/w -> Place ghost vertex  
rmb/e -> Place edge  


Shift + lmb/e -> Delete vertex  
-Vertex becomes ghost, ghosts gets deleted  
Shift+mmb/w -> Delete ghost vertex  
Shift+rmb/e -> Delete edge or chain place edge if currenly placing one  
r -> restart  
g -> toggle show ghost vertecies  
l -> toggle show grid lines  
h -> toggle show hitboxes on hover   


1-0 set color  
This applies to colors existing features and all new ones  
1: red  
2: orange  
3: yellow  
4: green  
5: blue  
6: purple  
7: pink  
8: white  
9: teal? custom color eventually  
0: black (default)  

Region colors are lighter than edge/vertex colors  
c -> color edge or vertex (whatever hovering over)  
Shift+c -> color region. Probably breaks something if you do it over a vertex.
